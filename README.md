## 📋 Overview

Implementation of Autoencoder (AE) and Variational Autoencoder (VAE) for
representation learning on Medical MNIST — a dataset of 6 anatomical regions:
AbdomenCT, BreastMRI, ChestCT, CXR, Hand, HeadCT.

A **separate AE and VAE** is trained for each region (12 models total).
